Emacs 24.3 on Windows 7
===

2014.03.24

### 1. How to use

git clone https://github.com/ansterd/emacs  
git submodule init  
git submodule update
```
(load "~/.emacs.d/init/init.el") ;; add to this line in yourr ~/.emacs
```
